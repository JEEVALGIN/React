React Full Roadmap (30 Minutes Daily) — Basic to Advanced
Designed for a learner who already knows some JavaScript. Estimated pace: 16 weeks at 30 minutes/day.

Phase 1 (Weeks 1–2): React Foundations
• How React works, component-based UI, SPA concept
• Setup: Node.js, npm, Vite, folder structure
• JSX syntax, expressions, fragments
• Functional components
• Props and reusable UI blocks
• Rendering lists, keys
• Conditional rendering
• Mini project: Todo UI

Phase 2 (Weeks 3–4): State & Events
• useState hook
• Handling events, forms, controlled inputs
• Two-way binding patterns
• Derived state, lifting state up
• Parent-child communication
• Mini project: Counter + Form Validator

Phase 3 (Weeks 5–6): Effects & Data
• useEffect lifecycle thinking
• Dependency array
• Fetching APIs with fetch/axios
• Loading, error, empty states
• Cleanup functions, timers
• Mini project: Weather/Search App

Phase 4 (Weeks 7–8): Routing & Styling
• React Router: routes, params, nested routes
• Navigation, protected routes basics
• CSS modules, Tailwind, styled-components overview
• Responsive UI
• Mini project: Multi-page Dashboard

Phase 5 (Weeks 9–10): Advanced State
• Context API
• useReducer
• Custom hooks
• Prop drilling vs context
• State libraries intro: Redux Toolkit, Zustand

Phase 6 (Weeks 11–12): Performance
• Re-render understanding
• memo, useMemo, useCallback
• Lazy loading, Suspense
• Code splitting
• List virtualization overview

Phase 7 (Weeks 13–14): Forms & Real Apps
• React Hook Form / Formik
• Validation with Zod/Yup
• Authentication flow basics
• CRUD apps
• Local storage/session storage

Phase 8 (Weeks 15–16): Advanced & Production
• Testing: Jest, React Testing Library
• Error boundaries
• Accessibility (a11y)
• Deployment: Vercel / Netlify
• SSR/SSG intro with Next.js
• TypeScript with React basics

Daily 30-Minute Formula
• 20 min learning one concept
• 10 min coding practice
• Every Sunday: revise + mini project improvement

Important JavaScript Concepts to Master Alongside React
• ES6 modules, destructuring, spread/rest
• Array methods: map/filter/reduce
• Promises, async/await
• Closures, scope
• this (basic understanding)
• Objects/immutability

Portfolio Projects (Build in Order)
• Todo App
• Movie Search App
• E-commerce Cart
• Notes App with local storage
• Admin Dashboard
• Chat App (Firebase/Supabase)

After Completion
• Learn Next.js deeply
• Learn TypeScript deeply
• Learn React Native
• System design for frontend
• Contribute to open source
Tip: Build while learning. React becomes easy only through projects.
